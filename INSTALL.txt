INSTALLATION
1. Extract this ZIP.
2. Copy the folders "assets" and "js" into the repository root.
3. Allow js/photo-fixes.js to replace the existing file.
4. Commit and push the changes to the main branch.
5. Wait for GitHub Pages to redeploy.
6. Hard-refresh the website with Ctrl+F5.
