# Third-party animation libraries

This website self-hosts two pinned, MIT-licensed animation libraries under `assets/vendor/`:

## AOS — Animate On Scroll

- Version: 2.3.4
- Repository: https://github.com/michalsnik/aos
- License: MIT
- Local files: `assets/vendor/aos/aos.css`, `assets/vendor/aos/aos.js`

## Anime.js

- Version: 3.2.2
- Repository: https://github.com/juliangarnier/anime
- License: MIT
- Local file: `assets/vendor/anime/anime.min.js`

The custom coffee bean rain, steam, menu styling and interaction code in `css/coffee-motion.css` and `js/coffee-motion.js` was written specifically for this website.
