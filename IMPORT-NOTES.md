# More Than a Kiosk — real photo import

Repository target: `alexzaxa/morethanaciosk`

## Included
- 15 professionally optimized real venue, drink, storefront and waterfront photos.
- 4 isolated Hand Pickers coffee-package images.
- Updated `js/coffee-motion.js` that automatically:
  - replaces the stock images in the homepage hero;
  - shows the four coffee packages as complete, uncropped products;
  - uses a 2×2 mobile layout so the package remains fully visible;
  - replaces the story and category images;
  - rebuilds the homepage gallery with all 15 real photos and lightbox support;
  - replaces the hero images on the Coffee, Cocktails, Kiosk and official Menu pages;
  - uses real storefront photos on Contact/Directions where the matching image element exists.

## Installation
Copy the contents of this ZIP into the repository root and allow it to replace:

`js/coffee-motion.js`

The new folder must remain at:

`assets/images/real-2026/`

Then commit and push the changes to the `main` branch.
